<h1>KoDansu</h1>

Inspired from *kona_dancer* by *Izya* released in May 12, 2007, KoDansu is a modern recreation of the app with Godot.  
Light and extensible.

![preview.gif](preview.gif)

## Features
- Resize via scroll wheel
- Right click opens `Options`
    - Access `Minimize`, `About`, and `Exit`

## Editing
1. To add a new GIF make a new AnimatedTexture resource first.
    - (see `Resources/GDResources/KonataDance.tres`)
2. Open your art software and export your GIF into individual frames.
3. Set each frame to the AnimatedTexture resource and save it.
